# Staples Education

A minimal, fast, and high-contrast landing page for Staples Education.

## Features
- **Design:** Deep royal purple and matte black color palette for a sophisticated, premium feel.
- **Hero Section:** Highlights the "Math Confidence Reset" framework with a clear Call To Action.
- **Brand Identity:** Clean, mathematical Capital Sigma ($\Sigma$) icon using a standard serif font.
- **Responsiveness:** Fully responsive design adapting beautifully from mobile to large desktop screens.

## File Structure
- `index.html`: The main structural entry point.
- `style.css`: All the styling rules and color palette.
- `README.md`: This documentation file.

## Setup
Simply open `index.html` in any modern browser to view the site. No build process required.

## Customization
To set up your live referral link, edit `index.html` and replace `[Insert your referral link here]` with your actual Wyzant referral URL inside the primary Call To Action button.
