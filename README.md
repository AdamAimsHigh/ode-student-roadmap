# ODE Student Roadmap

This repository contains the interactive pedagogical study guide and roadmap for Ordinary Differential Equations.

## Structure
- `Ordinary Differential Equations Study Roadmap.html`: The primary dynamic dashboard containing the video syllabus, micro-practice quizzes, and interactive visualizers.
- `index.html`: Web entry point for GitHub Pages.
- `ode_sorter.py`: The core python orchestrator that injects JSON video lists, quizzes, and scripts into the HTML templates.
- `generate_micro_quizzes.py`: Generates the LaTeX math quizzes via AI.
- `ODE-Manager-Local`: Contains data syncing scripts and local caches.

This project serves as a context anchor for autonomous development agents.
