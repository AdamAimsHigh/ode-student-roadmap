\section*{Part II --- Review Guide: Practice Problems}

Work the following ten problems in order. They climb from reading a slope field,
through the phase plane and the existence--uniqueness test, into fixed points,
contraction mappings, and a full Picard iteration. Solutions follow in Part III.

\begin{enumerate}[leftmargin=2em]
    \item For the slope field of $\dfrac{dy}{dx} = x + y$, find the slope at the
          point $(1,2)$ and the isocline along which every segment is
          horizontal.

    \item For $\dfrac{dy}{dx} = y^{2} - x$, find the curve along which every
          field segment has slope $2$.

    \item For $\dfrac{dy}{dx} = -\dfrac{x}{y}$, find the slope of the segment at
          $(3,4)$, and describe the shape of the integral curves.

    \item For the system $x' = y$, $y' = -x$, locate the equilibrium and
          classify it using the eigenvalues of its coefficient matrix.

    \item For the linear system $x' = 2x$, $y' = 3y$, write the eigenvalues of
          the coefficient matrix and classify the origin.

    \item For $\dfrac{dy}{dx} = \dfrac{x}{y - 2}$, determine where the existence
          and uniqueness theorem fails to guarantee a unique solution.

    \item Show that the initial value problem $\dfrac{dy}{dx} = \sqrt{y}$,
          $y(0) = 0$ has more than one solution, and explain which hypothesis of
          the theorem fails.

    \item Find every fixed point of $g(x) = x^{2}$ and classify each as
          attracting or repelling.

    \item Show that $g(x) = \cos x$ is a contraction on $[0,1]$, and state what
          the Banach fixed point theorem then concludes.

    \item \textbf{(Capstone)} For $\dfrac{dy}{dx} = y$ with $y(0) = 1$, carry out
          Picard iteration from $\varphi_0 = 1$: compute $\varphi_1$,
          $\varphi_2$, $\varphi_3$, and identify the function the iterates
          converge to.
\end{enumerate}

\section*{Part III --- Complete Solutions}

\noindent\textbf{Solution 1.}\quad The slope is just $f$ evaluated at the point.
\[ \textcolor{workpurple}{f(1,2) = 1 + 2 = 3.} \]
The horizontal segments lie on the isocline $f = 0$:
\[ \textcolor{workpurple}{x + y = 0 \;\Longrightarrow\; y = -x.} \]

\medskip
\noindent\textbf{Solution 2.}\quad Set $f$ equal to the target slope and solve.
\[ \textcolor{workpurple}{y^{2} - x = 2 \;\Longrightarrow\; x = y^{2} - 2.} \]
Every segment crossing the parabola $x = y^{2} - 2$ has slope $2$.

\medskip
\noindent\textbf{Solution 3.}\quad Substitute, keeping the leading sign:
\[ \textcolor{workpurple}{f(3,4) = -\frac{3}{4}.} \]
The integral curves satisfy $y\,dy = -x\,dx$, so
$\textcolor{workpurple}{\frac{y^2}{2} = -\frac{x^2}{2} + C_1}$, that is
\[ \textcolor{workpurple}{x^{2} + y^{2} = C} \qquad\text{(concentric circles).} \]

\medskip
\noindent\textbf{Solution 4.}\quad Equilibria need both derivatives zero:
\[ \textcolor{workpurple}{y = 0 \text{ and } -x = 0 \;\Longrightarrow\; (0,0).} \]
The coefficient matrix
$\left(\begin{smallmatrix}0&1\\-1&0\end{smallmatrix}\right)$ has characteristic
equation $\lambda^2 + 1 = 0$:
\[ \textcolor{workpurple}{\lambda = \pm i \;\;(\text{purely imaginary}).} \]
Purely imaginary eigenvalues give a \textcolor{workpurple}{center}: trajectories
are closed orbits circling the origin.

\medskip
\noindent\textbf{Solution 5.}\quad The system is decoupled, so the coefficient
matrix $\left(\begin{smallmatrix}2&0\\0&3\end{smallmatrix}\right)$ is diagonal
and its eigenvalues sit on the diagonal:
\[ \textcolor{workpurple}{\lambda_1 = 2, \quad \lambda_2 = 3.} \]
Both are real and positive, so the origin is an
\textcolor{workpurple}{unstable node (a source)} --- every nearby trajectory
flows outward.

\medskip
\noindent\textbf{Solution 6.}\quad Locate where $f$ and $\frac{\partial f}{\partial y}$
lose continuity. With $f = \dfrac{x}{y-2}$,
\[ \textcolor{workpurple}{\frac{\partial f}{\partial y} = -\frac{x}{(y-2)^2},} \]
and both blow up when the denominator vanishes:
\[ \textcolor{workpurple}{y - 2 = 0 \;\Longrightarrow\; y = 2.} \]
The theorem fails to guarantee uniqueness along the line $y = 2$; everywhere
else a unique solution exists.

\medskip
\noindent\textbf{Solution 7.}\quad Here $f = \sqrt{y}$ is continuous at $y=0$,
but
\[ \textcolor{workpurple}{\frac{\partial f}{\partial y} = \frac{1}{2\sqrt{y}}} \]
is \emph{not} continuous at $y = 0$ --- it blows up. So uniqueness is not
guaranteed, and indeed two solutions pass through the origin:
\[ \textcolor{workpurple}{y = 0} \qquad\text{and}\qquad
   \textcolor{workpurple}{y = \frac{x^{2}}{4}} \quad (x \ge 0), \]
since $\frac{d}{dx}\!\left(\frac{x^2}{4}\right) = \frac{x}{2} = \sqrt{x^2/4}$.
The failed hypothesis is continuity of $\frac{\partial f}{\partial y}$.

\medskip
\noindent\textbf{Solution 8.}\quad Solve $g(x) = x$:
\[ \textcolor{workpurple}{x^{2} = x \;\Longrightarrow\; x(x-1) = 0 \;\Longrightarrow\; x = 0,\ 1.} \]
Classify with $|g'(x)| = |2x|$:
\[ \textcolor{workpurple}{|g'(0)| = 0 < 1 \ (\text{attracting}), \qquad |g'(1)| = 2 > 1 \ (\text{repelling}).} \]

\medskip
\noindent\textbf{Solution 9.}\quad A bound $|g'(x)| \le k < 1$ on an interval
makes $g$ a contraction there. With $g(x) = \cos x$,
\[ \textcolor{workpurple}{|g'(x)| = |\sin x|,} \]
which increases on $[0,1]$ and so is largest at $x = 1$:
\[ \textcolor{workpurple}{|\sin x| \le \sin 1 \approx 0.841 < 1.} \]
Thus $g$ is a contraction with $k = \sin 1$. By Banach, $g$ has a
\textcolor{workpurple}{unique} fixed point in $[0,1]$, and iterating cosine from
\emph{any} start converges to it.

\medskip
\noindent\textbf{Solution 10.}\quad Use $\varphi_{n+1}(x) = 1 + \int_0^x \varphi_n(t)\,dt$
from $\varphi_0 = 1$.
\[ \textcolor{workpurple}{\varphi_1 = 1 + \int_0^x 1\,dt = 1 + x.} \]
\[ \textcolor{workpurple}{\varphi_2 = 1 + \int_0^x (1+t)\,dt = 1 + x + \frac{x^2}{2}.} \]
\[ \textcolor{workpurple}{\varphi_3 = 1 + \int_0^x \left(1 + t + \frac{t^2}{2}\right)dt = 1 + x + \frac{x^2}{2} + \frac{x^3}{6}.} \]
These are the partial sums of the exponential series, so
\[ \textcolor{workpurple}{\varphi_n \to 1 + x + \frac{x^2}{2} + \frac{x^3}{6} + \cdots = e^{x},} \]
the true solution of $y' = y$, $y(0) = 1$.

\vspace{0.8em}
\begin{conceptbox}[Unit 3 Key Takeaway]
This unit reads differential equations as \textcolor{accentorange}{geometry and
guarantees} rather than formulas. A \textcolor{accentorange}{slope field} turns
$y'=f(x,y)$ into a picture its integral curves follow; the
\textcolor{accentorange}{phase plane} does the same for systems. The
\textcolor{accentorange}{existence and uniqueness theorem} ($f$ and
$\partial f/\partial y$ continuous) says exactly one curve threads each point,
and \textcolor{accentorange}{Picard iteration via the Banach contraction
principle} is the machine that proves it.
\end{conceptbox}
